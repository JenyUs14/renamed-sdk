# OpenAPI Specification

This directory will contain the OpenAPI specification for the renamed.to API.

## Status

The OpenAPI spec is maintained in the main renamed.to project and will be synced here.

## Usage

Once available, you can use the spec to:
- Generate additional SDK clients
- Validate API responses
- Generate API documentation

## Manual Reference

Until the OpenAPI spec is available, refer to:
- [API Documentation](https://www.renamed.to/docs/api-docs)
- Individual SDK implementations in `../sdks/`
